# Cebian 1.2.1 Release Notes

## 产品定位

**边栏（Cebian）** 将四类能力整合在一款应用中：

- 👆 **边缘手势** — 侧滑触发条、48 种系统动作、快速启动器、任务切换器
- 🔮 **悬浮球** — 屏幕取词、文字/图片搜索、翻译、区域截图与 OCR
- 🔔 **消息中心** — 通知历史、消息提醒样式、OTP/验证码提取与自动填充
- 🧩 **扩展工具** — Shell 命令面板、悬浮指针、Widget 面板、ZIP 设置备份

## 本版本亮点

- **ZIP 设置备份**：导出/导入 `cebian-backup-*.zip`，包含 `settings.json`（format v2）及搜索引擎图标、悬浮球资源、钉图暂存；可选包含 OTP 记录与通知历史；导入前预览确认
- **悬浮球取词**：摇杆 + 全屏指针取词，无障碍优先、OCR 降级、聚合以图搜图与可配置文字搜索引擎
- **悬浮指针**：虚拟摇杆、径向功能环、边缘触发、手势录制与回放
- **扣桌手势**：亮屏时屏幕朝下平放静止后触发（默认锁屏并静音响铃）
- 底部导航「消息」Tab：通知历史、消息提醒与 OTP 集中入口
- OTP 自动填充成功率统计；可选 LSPosed 短信捕获与系统注入
- Shell 命令模板变量：`{package}`、`{foreground_package}`、`{timestamp}`、`{datetime}` 等
- 应用内隐私政策与 Special Use 前台服务说明

## 隐私与数据

通知、OTP 与设置数据默认仅保存在本机。以图搜图时图片可能临时上传至第三方图床或搜索引擎；ZIP 备份文件由用户自行保管。详见应用内 **扩展 → 关于 → 隐私政策**。
